# Portfolio Website README

This is a React-based portfolio website styled with Vanilla CSS, copying the design provided.

## Project Structure

- `src/components`: Contains individual section components (Hero, Projects, Experience, Toolbox, Contact).
- `src/data.js`: Contains the content for the website (Projects data, experience, etc.).
- `src/index.css`: Global styles and variables.
- `src/App.jsx`: Main entry point assembling the sections.

## How to Run

1. Navigate to the directory:
   ```bash
   cd portfolio
   ```

2. Install dependencies (already installed):
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## Customization

- To update the content (text, projects, stats), edit `src/data.js`.
- To update images, replace the file in `public/profile.jpg` or modify the components.
- To update styles, edit the corresponding `.css` files in `src/components/` or `src/index.css` for global themes.
